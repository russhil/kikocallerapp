🛠 Facing Issues? Try This समस्या आ रही है? इसे ट्राई करें

⚠️
Issue: "I am not able to view orders after the call" समस्या: "कॉल के बाद मुझे ऑर्डर नहीं दिख रहे हैं"
Step A: Check Basics स्टेप A: बेसिक्स चेक करें
  • Ensure Android version is v13 or above.
  • सुनिश्चित करें कि आपका फ़ोन Android v13 या उससे ऊपर का है।
  • Check if call recording is active in your dialer.
  • चेक करें कि आपके डायलर में कॉल रिकॉर्डिंग चालू है या नहीं।
  • Verify if old recordings exist in your file manager.
  • फ़ाइल मैनेजर में चेक करें कि पुरानी रिकॉर्डिंग मौजूद हैं या नहीं।
  • If not → your phone's call recording is NOT working.
  • यदि नहीं → तो आपके फ़ोन की कॉल रिकॉर्डिंग काम नहीं कर रही है।
Step B: If Recording Not Supported स्टेप B: अगर रिकॉर्डिंग सपोर्ट नहीं करती
  • Install O Dialler from Play Store.
  • Play Store से O Dialler ऐप इनस्टॉल करें।
  • Works best on: OnePlus, Vivo, Oppo.
  • यह इन फ़ोन में अच्छा काम करता है: OnePlus, Vivo, Oppo.
  • Make a test call to see if recordings are saved in the folder.
  • एक टेस्ट कॉल करें और चेक करें कि रिकॉर्डिंग फ़ोल्डर में सेव हो रही है या नहीं।
Step C: Folder Check स्टेप C: फ़ोल्डर चेक करें
  • Ensure the correct recording folder is selected inside the Kiko app settings.
  • सुनिश्चित करें कि किको ऐप की सेटिंग्स में सही रिकॉर्डिंग फ़ोल्डर चुना गया है।
Step D: Default App Check स्टेप D: डिफ़ॉल्ट ऐप चेक करें
  • Ensure Kiko is set as the default caller ID app.
  • सुनिश्चित करें कि किको को डिफ़ॉल्ट कॉलर आईडी ऐप (Default Caller ID) के रूप में सेट किया गया है।
👉 If still not working: Try using another mobile device (some phones don’t support AI processing properly). 👉 अगर फिर भी काम नहीं कर रहा है: तो किसी दूसरे मोबाइल डिवाइस का उपयोग करके देखें (कुछ फ़ोन AI को ठीक से सपोर्ट नहीं करते हैं)।
